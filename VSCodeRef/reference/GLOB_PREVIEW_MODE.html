GLOB_PREVIEW_MODEtype of current preview (view dependent, do not use in parameter/property scripts)

  • 0 - None

  • 1 - Dialog

  • 2 - Listing

  • 3 - Favorite saving

Use the exact needed values. Using ranges are not recommended due to possible future value extensions.
