ROUND_INT

ROUND_INT (x)

Returns the rounded integer part of x. The 'i = ROUND_INT (x)' expression is equivalent with the following script:

IF x < 0.0 THEN i = INT (x - 0.5) ELSE i = INT (x + 0.5)
