BRICK

BRICK a, b, c

The first corner of the block is in the local origin and the edges with lengths a, b and c are along the x-, y- and z-axes, respectively. Zero values create degenerated blocks (rectangle or line). 

a >= 0, b >= 0, c >= 0
a + b + c > 0
