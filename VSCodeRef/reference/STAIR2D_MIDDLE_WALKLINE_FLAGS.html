STAIR2D_MIDDLE_WALKLINE_FLAGSinteger array with two dimensions ([n][2]), additional data of stair walking line nodes, middle part
2D
3D
UI
Parameter
Property
Default
[0]

n = number of polyline nodes for the middle part of the walking line

mask: returns information about the node position
mask = j1 + 2*j2 + 4*j3 + 8*j4, where each j can be 0 or 1.
j1: node is on trail end of first tread
j2: node is on lead end of landing
j3: node is on trail end of landing
j4: node is on lead end of last tread

  • [n][1] - location mask of the node

  • [n][2] - index of tread where the node is on trail end or above
