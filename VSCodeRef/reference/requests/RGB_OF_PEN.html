RGB_OF_PEN

n = REQUEST("RGB_OF_PEN", penindex, r, g, b)

Like the REQ function (but in just one call), returns in the specified variables the value of the r, g, b components of the pen. Expression returns 0 containing dummy return values (emtpy string or 0) if used in parameter script, causing additional warning.
