MATCHING_PROPERTIES

n = REQUEST("MATCHING_PROPERTIES", type, name1, name2, ...)
type: Selects which property library parts to request
1: individually associated.
otherwise: associated by criteria.
namei: Returned library part names.

If used in an associative label, the function returns the properties of the element the label is associated with. Expression returns 0 and contains dummy return values (emtpy string or 0) if used in parameter script, causing additional warning.
