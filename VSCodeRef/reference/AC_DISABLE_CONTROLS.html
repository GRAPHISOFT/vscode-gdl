ac_disable_controlsinteger

This parameter can control the visibility of the Font Size ("ROOM_LSIZE") input area of the Zone Stamp settings dialog: 0 or the object doesn't have the parameter - show Font Size, 1 - hide Font Size (therefore allowing extra space for the parameter list)

Compatibility: inactive since ARCHICAD 22. The "Font Size" input control has moved to the "Text Style" panel, its visibility is controlled by LOCK/HIDEPARAMETER commands.
