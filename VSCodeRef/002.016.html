Chapter 13. New GDL Features in Archicad 27

This section contains a fly-through of the new GDL features in Archicad 27.

New MEP tools

Some of the MEP tools got united under MEP Routing tools. Duct Routing, Pipe Routing, Cable Carrier Routing each have a straight, elbow and transition subelement, with some common attribute settings. These are returned by the "MEP_ROUTE_ATTRIBUTES" REQUEST. Corresponding "MEPSYSTEM" APPLICATION_QUERYs can be used only with non-routing MEP tools' elements.

New built-in properties

Design option-related built-in properties are available with a fixed name for GDL:

  • Builtin.Design_Option_Name

  • Builtin.Design_Option_ID

  • Builtin.Design_Option_Set_Name
